# Sass Numeric Functions
* The numeric functions are used to manipulate numeric values.

## Numeric Functions List
* abs(number)
* cail(5.6)  
* comparable(num1, num2)
* floor(5.1)
* max(1, 2, 10, 15, 18)
* min(2, 10, 9, 1, 6)
* percentage(number) `(Multiply the number with 100)`
* random ()
* round(15.80) `return 16`
